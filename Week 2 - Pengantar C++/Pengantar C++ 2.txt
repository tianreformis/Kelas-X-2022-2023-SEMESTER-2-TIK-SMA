Tipe Data, pengelompokkan data/pembagian data
Variabel, peubah, identifier mewakili tipe data.

Tipe Data Primitif :
int = bilangan bulat
float/double = bilangan cacah, bilangan pecahan.
char = karakter.
string = kalimat 
boolean = hnya bernilai true dan false 


Tipe Data Jamak/Majemuk :
array = tipe data majemuk yang mewakili satu jenis tipe data
record = tipe data majemuk yang mewakili banyak jenis tipe data

Header, bisa disebut juga dengan library digunakan untuk memanggil function - function lainnya.
Dimulai dengan #include<> 
1. Header iostream (input output stream), terdapat function cout (untuk membuat output) dan cin (untuk membuat input).
2. Header math, untuk function matematis.
3. Header stdio (standard input output), terdapat function printf (untuk membuat output) dan scanf untuk membuat input),
